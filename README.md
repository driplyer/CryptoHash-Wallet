# CryptoHash-Wallet

## 🛠️ Configuration requise et installation

Ce projet utilise **Gradle 9.5.0**, ce qui impose l'utilisation d'une version spécifique du kit de développement Java (JDK) pour compiler et synchroniser le projet sans problème.

### 1. Prérequis

* **Version du JDK :** Java 21 (ou supérieure)
* **Version d'Android Studio :** Android Studio Ladybug (2024.2.1) ou plus récent est fortement recommandé pour garantir la compatibilité avec Gradle 9.X.X

### 2. Configuration requise dans l'IDE (Windows)

* **Pour éviter les échecs de synchronisation ou les erreurs vous devez configurer explicitement Android Studio pour utiliser le JDK 21 :

1. Ouvrez le projet dans **Android Studio**.
2. Allez dans le menu : **File** ➔ **Settings...** (ou utilisez le raccourci `Ctrl + Alt + S`).
3. Dans la colonne de gauche, naviguez vers : **Build, Execution, Deployment** ➔ **Build Tools** ➔ **Gradle**.
4. Repérez le menu déroulant nommé **Gradle JDK**.
5. Sélectionnez **JetBrains Runtime 21** (ou téléchargez/sélectionnez votre installation locale de JDK 21).
6. Cliquez sur **Apply**, puis sur **OK**.
7. Lancez la synchronisation en cliquant sur le bouton **Sync Project with Gradle Files** (l'icône d'éléphant 🐘) dans la barre d'outils supérieure.
